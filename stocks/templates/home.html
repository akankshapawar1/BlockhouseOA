Welcome to the Stock API Service

Here are the available API endpoints:

1. Fetch Stock Data

Use the following endpoint to fetch stock data for a given symbol:

GET /fetch/<symbol>/

Example: GET /fetch/AAPL/

2. Run Backtest

Use the following endpoint to run a backtesting strategy:

GET /backtest/?symbol=<symbol>&initial_investment=<amount>&short_window=<short_window>&long_window=<long_window>

Example: GET /backtest/?symbol=AAPL&initial_investment=10000&short_window=50&long_window=200

3. Get Stock Predictions

Use the following endpoint to get stock price predictions for the next 30 days:

GET /predict/?symbol=<symbol>

Example: GET /predict/?symbol=AAPL

Note: This will always return predictions for the next 30 days.

4. Generate Report (PDF/JSON)

Use the following endpoint to generate a report in PDF or JSON format:

GET /report/<symbol>/?format=<pdf|json>

Example for PDF: GET /report/AAPL/?format=pdf

Example for JSON: GET /report/AAPL/?format=json
